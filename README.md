### DEV Environment Setup:

  #### Variables:

   - **%FS_FRONTEND_HOME%:** The root path of your FP front-end codes

   - **%FS_RESOURCE_HOME%:** The root path of your FP resource files
   
  #### Front-end
  
   npm install
  
  #####  RUN IN DEV
  
   npm start
  
  #####  RUN IN PROD
  
   npm run build:prod
